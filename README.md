**Info Ciledug** adalah sebuah portal informasi satu halaman (*Single Page Portal*) yang modern, interaktif, dan responsif. Dirancang khusus untuk warga Ciledug dan sekitarnya guna mendapatkan informasi terkini mulai dari berita lokal, rekomendasi kuliner, hingga eksplorasi wilayah dalam satu tampilan yang premium.

## ✨ Fitur Unggulan

- 📱 **Desain Ultra-Responsif**: Pengalaman pengguna yang mulus baik di smartphone maupun desktop.
- 🎥 **Integrasi Instagram Reels**: Menampilkan konten video Reels secara langsung dengan optimasi *Lazy-Loading* (Hemat kuota & Performa kencang).
- 🕒 **Informasi Metadata**: Dilengkapi dengan label kategori, deskripsi, hashtag, dan informasi waktu postingan.
- 🗺️ **City Explorer**: Integrasi Google Maps untuk memudahkan pencarian lokasi strategis di wilayah Ciledug.
- ⚡ **Performa Tinggi**: Dibangun dengan Next.js 15+ dan Turbopack untuk kecepatan loading yang maksimal.
- 🎨 **Branding Kustom**: Logo dan favicon yang unik sesuai identitas Info Ciledug.

## 🚀 Cara Menjalankan Proyek

### 1. Persiapan
Pastikan Anda sudah menginstal [Node.js](https://nodejs.org/) dan [pnpm](https://pnpm.io/).

### 2. Instalasi Dependensi
```bash
pnpm install
```

### 3. Menjalankan Mode Pengembangan
```bash
pnpm dev
```
Buka [http://localhost:3003](http://localhost:3003) di browser Anda.

### 4. Membangun Versi Produksi
Anda bisa menggunakan file otomatis yang sudah disediakan:
- Klik 2x pada file `release.bat` untuk membangun dan menjalankan versi produksi secara instan.

## 🛠️ Teknologi yang Digunakan

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS 4
- **Animation**: Framer Motion
- **Iconography**: Custom Assets & Lucide Icons
- **Deployment**: Optimized for Vercel/Production servers
